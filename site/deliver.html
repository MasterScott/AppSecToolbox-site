Deliver Tools

Stage 4 of your AppSec Pipeline

The forth and final stage of an AppSec Pipeline which collects and normalizes the data created during testing. Any duplicate findings should be removed so that the same issue found by multiple tools is only reported once. Here we link to issue tracking systems, produce reports, and otherwise provide data for stakeholders.

Purpose of this site...

The purpose of this site is to provide unbiased information on tools that can be used by AppSec programs to automate the assessment of applications as much as possible. The goal is to provide an ala carte menu of tools so that AppSec program teams can select those that work best for their specific situation. For more information, see:
